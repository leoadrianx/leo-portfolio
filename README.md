### Prerequisites

- Node.js (v16 or higher)
- npm or yarn package manager

### Setup

1. Clone the repository:
```bash
git clone https://github.com/leoadrianx/leo-portfolio.git
cd leo-portfolio
```

2. Install dependencies:
```bash
npm install
```

3. Start the development server:
```bash
npm run dev
```

The application will be available at `http://localhost:5173`

## ğŸ¨ Customization

### Update Portfolio Content

Edit the data files in `src/data/` to customize:

- Personal information
- Skills and experience
- Projects and portfolio items
- Social links

### Modify Styling

- **Global Styles**: Edit `src/index.css` for global TailwindCSS utilities
- **Component Styles**: Use TailwindCSS classes in your JSX components
- **Custom CSS**: Add custom CSS in component-level CSS files

### Add New Sections

1. Create a new component in `src/components/`
2. Create a section in `src/sections/`
3. Import and use it in `src/App.tsx`
